Beschreibung:
-------------
FlashGamesrr erlaubt es Flashgames auf der eigenen Seite einzubauen. Dies funktioniert ganze einfach indem man den Link des FlashGames im Adminbereich einbindet.
um FlashGames runterzuladen schaut mal hier:
http://www.wikihow.com/Download-a-Flash-Game


Entwickelt
----------
° FeTTsack
° auf Basis von IlchClan 1.1 o


Installation:
-------------
° Backup machen
° alle Dateien im Ordner upload, in ihrer Ordnerstrucktur hochladen
° installation.php ausführen 
	bsp: www.deineSeite.de/index.php?installation
